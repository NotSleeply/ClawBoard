## 需求描述

为 ClawBoard 添加运行时健康监控功能，在设置面板或统计面板中显示系统资源使用情况。

## 功能需求

- **内存使用量** - 显示当前进程内存占用（MB）
- **CPU 占用** - 显示当前进程 CPU 使用率（%）
- **数据库大小** - 显示数据库文件大小
- **记录总数** - 显示当前剪贴板记录数
- **运行时长** - 显示应用启动以来的运行时长

## UI 设计

- 在统计面板底部添加 "系统状态" 区域
- 使用进度条显示内存和 CPU 占用
- 实时刷新（每 5 秒更新一次）
